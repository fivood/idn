# 侦查控制台 (Detective Console)

一个结合侦查技能升级与双语推理小说阅读的挂机增量游戏。

## 项目简介

玩家通过升级侦查助手、科学仪器等获取侦查经验（DI）。DI 可用于升级技能或自动破译小说段落。随着小说段落的解密，玩家可阅读中英对照文本、发掘案情线索、查看嫌疑人状态。在全案告破后，可支付 DI 结案并将其归档至书库中随时翻阅。

## 系统特性

1. 侦探分组与小说数据对齐
   - 案卷按主要侦探角色进行分组展示（如经典悬疑、丹尼尔·霍桑系列）。
   - 支持《无人生还》、《关键词是谋杀》与《关键句是死亡》三部小说的章节及中英段落双语对齐。

2. 挂机增益与生涯荣誉
   - 结案归档的小说将收录至侦探书库，每部已结案卷会提供 +20% 的全局 DI 产量加成与 +10% 的自动解密速率提升，增益可在多周目中累加。

3. 极简进度条
   - 采用 3px 宽度的细线进度条，无文字展示，放置在页眉与导航栏之间。进度条颜色根据解密效率在不同主题下自动调整。

4. 界面版面
   - 采用 7:3 左右分栏布局。左侧大分栏展示案卷、嫌疑人及解密进度，右侧展示技能升级项。
   - 顶部提供扁平化的案情备忘录与生涯荣誉面板，降低纵向空间占用。

## 项目部署与运行

### 目录结构
- parse_all_epubs.cjs：小说双语 epub 提取与对齐编译脚本。
- src/App.jsx：核心游戏逻辑与状态管理。
- src/components/Console.jsx：控制中心与升级面板。
- src/components/NovelWorkspace.jsx：侦查工作区（嫌疑人、线索、自动阅读器）。
- src/components/Library.jsx / BookReader.jsx：已结案卷书库与对照阅读器。
- src/index.css：全局样式规范。

### 本地启动
1. 安装依赖：
   ```bash
   npm install
   ```
2. 启动开发服务器：
   ```bash
   npm run dev
   ```
3. 构建打包：
   ```bash
   npm run build
   ```
