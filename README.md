# 贝克街档案柜

一个融合推理小说双语阅读与挂机增量解密玩法的网页与桌面游戏。

[在线游玩](https://idn.fivood.com) | [桌面端下载](https://github.com/fivood/idn/releases/latest)

---

## 游戏机制

玩家通过升级侦查助手、科学仪器等装备积累侦查经验值（DI），利用 DI 自动破译推理小说的原文密文段落。

在破译推进的过程中，玩家可以：
- 逐页阅读中英双语对照原典。
- 发掘案情线索，并在独立软木板线索墙上拖拽、连线推理。
- 追踪嫌疑人动态，根据阅读进度动态解锁角色信息（防剧透）。
- 指控幕后真凶，答对赢得 DI 奖励。
- 结案归档以获得全局 DI 产量与解密速度的永久生涯加成。

## 本地开发部署

```bash
# 克隆仓库
git clone https://github.com/fivood/idn.git
cd idn

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 项目结构

- `public/data/` - 按需动态载入的单本小说正文数据。
- `src/App.jsx` - 核心状态引擎、挂机循环与存档逻辑。
- `src/components/` - 界面组件（Library, NovelWorkspace, ClueWallModal, BookReader）。
- `src/data/` - 小说配置、嫌疑人、线索墙及结构元数据。
- `src-tauri/` - Tauri 桌面端封装配置。
- `parse_all_epubs.cjs` - EPUB 双语段落提取对齐编译脚本。
