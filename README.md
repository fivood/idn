# 侦查控制台 (Detective Console)

> **“在探求真相的道路上，每一次结案都是您荣誉章上的新刻痕。已归档的案件将为您源源不断地提供声望与解密直觉。”** —— 侦查助手

---

## 🔍 项目简介

本项目是一款将**专业侦查技能升级**与**双语经典小说文本解密**深度交织的极简主义挂机增量（Idle/Incremental）探案游戏。

侦探通过配置与升级侦查助手、科学仪器（如放大镜、法医试剂箱、电报监听等），自动累积**“侦查经验 (Detective Intelligence, DI)”**。DI 可用于进一步升级技能，或是对“未决案卷”中被加密的小说段落进行实时破译。随着解密的推进，侦探可以探索线索、查验嫌疑人死伤、参与断案，并在全案告破后支付 DI 点数“结案归档”，永久收录至“侦探书库”中进行自由翻阅。

---

## 🎨 核心重构与系统特性

### 1. 🗂️ 侦探分组与三书编译对齐系统
* **侦探分组归类**：游戏中的案卷自动以主要探案角色（如 `☉ 侦探：丹尼尔·霍桑` 与 `☉ 侦探：经典悬疑 (无固定侦探)`）进行聚合分组陈列，逻辑清晰。
* **小说数据库对齐**：支持将阿加莎·克里斯蒂的《无人生还》、安东尼·霍洛维茨的霍桑系列《关键词是谋杀》与《关键句是死亡》这 3 部小说进行章节、中英段落双语对齐，并自动编译输出为底层的对齐段落数据库。

### 2. 🎖️ 双层挂机增益与生涯荣誉系统 (方案 C)
* **全局生涯加成**：每次将一部小说完成全部阅读并支付 DI 进行“结案归档”后，该案件会被永久收录至“侦探书库”。
* **荣誉机制**：每宗已结案卷会为侦探提供 **+20% 的全局 DI 产量加成** 与 **+10% 的自动解密速率提升**。增益终身继承并跨案卷累加，构成了深度的多周目流转机制。

### 3. ⏳ 3px 极简流光进度条与双色反转色表
* **3px 极简线条**：删去了进度条中繁冗的数值文字，将其压缩为极简的 `3px` 流线，置于顶栏 `Header` 和选项卡 `Tabs` 之间，具有呼吸式的霓虹投影边缘。
* **周期分段装载**：避免高产出阶段进度条闪烁。系统将单次周期恒定在 1~2.5 秒左右，高产出时自动将产出以 `1`, `2`, `5`, `10`, `20`, `50`, `100`, `500` 等清晰整数块的形式周期性装载累加。
* **冷色卡对比度反转**：采用定制的石板灰-蓝绿（Slate-Teal）冷色调渐变，并支持双色主题的自适应反转：
  - **浅色模式下**：从浅到深（`Bright Star #e7ede6` $\rightarrow$ `Full Moon Sky #293535`），速度越快颜色越深以维持与亮色背景的对比度。
  - **深色模式下**：从深到浅（`Full Moon Sky #293535` $\rightarrow$ `Bright Star #e7ede6`），速度越快颜色越亮以形成黑暗环境下的荧光律动线。

### 4. 🎛️ 7:3 扁平化仪表盘布局
* **列排版调换**：小说案卷移至左侧大分栏，升级项目移至右侧窄分栏，更契合从左向右的视觉阅读路线。
* **70% 宽的案卷列**：为信息密集的嫌疑人、线索、段落解密进度条提供充分的横向展示空间。
* **30% 宽的升级列**：针对窄列空间，将升级项卡片改造为精致的**垂直堆叠模式与 100% 满宽升级按钮**，在桌面和移动设备上皆极具几何美感。
* **扁平化顶部**：将“案情备忘录”和“侦探生涯荣誉”的 padding 收窄至 `6px 12px`，高度减半并左右并排，释放纵向高度，避免网页滚动。

## 🛠️ 项目部署与运行

### 1. 项目目录结构
* `parse_all_epubs.cjs`：多小说双语 epub 提取与对齐编译脚本。
* `src/App.jsx`：核心探案引擎与全局状态。
* `src/components/Console.jsx`：侦查大厅、案卷陈列、技能升级、生涯面板。
* `src/components/NovelWorkspace.jsx`：线索物证墙、嫌疑人板、打字机自动阅读室。
* `src/components/Library.jsx` / `BookReader.jsx`：书库与对照阅读器。
* `src/index.css`：黑白灰极简美学样式表。

### 2. 本地快速启动
1. 安装依赖项：
   ```bash
   npm install
   ```
2. 运行本地开发服务器：
   ```bash
   npm run dev
   ```
3. 打开浏览器访问默认的本地端口（通常为 `http://localhost:5173/`）。
4. 运行 `npm run build` 进行生产包打包构建，验证文件结构正确性。

## 🎨 极简 Noir 探案视觉规范
* **纯色底噪**：严格遵守黑、白、深灰的底色表现，去除带有现代情绪的彩底 Emoji。
* **克莱因蓝/冷青**：表示不容忽视的证据（如线索、升级）。
* **深红/暗红**：表示已故的死者或消逝的章节。
* **多端适配**：组件高度锁定，去除了页面外层滚动条，完美融入各种侧边栏办公窗口。
