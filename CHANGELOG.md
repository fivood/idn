# 贝克街私家侦探档案柜 - 更新日志 (Changelog)

所有本项目的版本更新历史和未来规划均记录于此。

---

## 📢 版本发版策略 (Release Strategy)

- **大更新 (Major/Minor Update)**：包含核心机制重构、新增案卷书籍、重大功能模块上线。此类更新将**直接触发版本升级并进行全平台构建发版**。
- **小更新 (Patch Update)**：包含常规 Bug 修复、文案微调、微小体验调优。此类小改动**平时仅提交至代码仓库，不立即触发发版**，累积到一定程度（或伴随大更新）时再合并统一发布新版本。

---

## 🗺️ 版本历史 (Version History)

### [v1.2.3] - 2026-06-05
> **本次更新重点：右下角遇害者人数防剧透优化与倍率进度条精简**

#### 🚀 新增与修复 (Features & Fixes)
- **优化右下角遇害者人数防剧透 (Victim Count Spoiler Prevention)**：
  - 移除了“遇害者人数”统计中的嫌疑人总数分母（如 `/ 10`）和对应的红色进度条，避免暴露全案最终遇害者数量。改为直接展示当前已发现的遇害者数量（如 `0 人`、`1 人`），彻底杜绝通过该进度面板产生的故事结局剧透。
  - 移除了《无人生还》专有的“玩偶全局倍率”进度条，避免通过固定分母（`10`）泄露小兵玩偶和遇难者的总数，仅保留纯文字倍率显示（如 `+30%`）。

### [v1.2.2] - 2026-06-05
> **本次更新重点：自动播放死锁修复、打字倍速调整支持、右栏空间优化布局与按钮字样精简**

#### 🚀 新增与修复 (Features & Fixes)
- **修复自动播放段落推进死锁 (Autoplay Fix)**：
  - 定位并解决了当后台挂机解密以 100ms 频率累加时间并重建 `novelState` 引用时，导致前台翻页与打字机 `useEffect` 挂钩每过 100ms 都会被强行清除重建的 Bug。
  - 移除了 React 挂钩依赖项中的 `novelState`，改用已结构化解耦的数字类型（`pagesRead`、`pagesUnlocked`）精确依赖。现在自动播放完毕后，能完美顺畅地在 1.2 秒后继续翻阅下一页。
- **新增播放倍速切换 (Autoplay Speed Controls)**：
  - 在控制栏引入了 `0.5x`、`1.0x`、`1.5x` 和 `2.0x` 四档速率调整，并支持将用户倍速选择持久化保存至 `localStorage`。
  - 速率会等比动态调整打字出字速度（最高达 10ms/字）与页间停留等待延迟，极大地提升了玩家快速推进剧情的可控性。
- **现场侦查右侧栏布局与按钮字样精简 (UI Layout)**：
  - 将“📌 案卷线索墙”按钮从左侧阅读器顶部，移出并移至右侧栏“当前侦查指令”面板上方，将其重塑为 `width: 100%` 的饱满大按钮，有效填补了由于阅读器高度坍塌修复产生的多余空白间隙。
  - 将右上角“中文”与“英文”切换按钮字样精简为“中”与“EN”，释放控制区排版空间。
  - 全案 5 部小说的开篇防剧透过滤逻辑也已同步通过倍速与严格大于号检验，开篇 0 进度下嫌疑人与物证卡片实现完美隐藏。

### [v1.2.1] - 2026-06-05
> **本次更新重点：小说文本数据动态分包与懒加载、窗口标题版本残留清理、自更新体验优化**

#### 🚀 新增与修复 (Features & Fixes)
- **小说数据动态分包与按需懒加载 (Performance & Size)**：
  - 将原先打包进主 JS 中重达 4.6 MB 的全部小说文本库 `novel_data.json` 进行了拆分。
  - 提取出仅保留章节结构、标题及页数计数的超轻量元数据 [novel_metadata.json](file:///d:/fukki/idn/src/data/novel_metadata.json)（仅 45 KB），静态打包供书架和挂机后台进行秒开结算。
  - 将每本小说的完整中英正文分别独立打包至 `public/data/` 目录中。当玩家点击“继续调查”或“取书阅读”时，前端使用 `fetch` 动态按需装载正文并缓存，加载期间展示简洁的等待界面。
  - **优化效果**：项目构建生成的主 JS 文件体积从原先的 **3.26 MB 骤降至 276 KB (缩减约 91.5%)**，首屏载入速度获得史诗级优化。
- **清理窗口标题栏版本残留 (UI)**：
  - 移除了 Tauri 桌面端窗口配置中硬编码的 `v1.0.0` 后缀，改为了自适应干净的 `贝克街私家侦探档案柜`。避免了客户端版本实际升级但标题栏依然显示 `v1.0.0` 的版本漂移不一致问题。
- **自更新体验优化与安全防误推机制**：
  - 优化并接入了自更新机制。当用户本地运行的是最新版本时，点击“检查更新”能获得清晰明确的 `✓ 您已在使用最新版本，无需更新。` 友好提示。
  - 更新了 `.gitignore`，增加了对 Tauri 签名生成私钥及公钥文件 `*.key` 和 `*.key.pub` 的自动忽略拦截，防范开发者误推送敏感安全凭据至远程仓库。
- **全屏自适应宽度布局改造 (UI Layout)**：
  - 移除了在大屏下强制锁定 16:9 画幅比例的高宽限制，使得主容器能够 100% 自适应填满浏览器视口或桌面端客户端窗口，杜绝空间浪费，为案件正文的双语对比展示提供更开阔的区域。
  - 去除了外侧在桌面端的双重嵌套黑边、圆角及投影样式，使 Tauri 桌面端窗口看起来更加干净清爽。



### [v1.2.0] - 2026-06-05
> **本次更新重点：玩偶机制融入、章节菜单回顾、物证线索升级合入线索墙、侧边栏详情长驻与自适应等比缩放板面**

#### 🚀 新增特性 (Features)
- **线索详情侧边栏长驻**：
  - 移除了原先的多重遮罩 Modal 弹窗交互，改为案卷墙左右分栏布局。左侧为等比自适应的大板，右侧长驻精美公文袋详情栏，并支持在点击节点时动态显示其摘要描述、双语口供、遇害死因等信息。
- **十个小士兵歌词纸内聚**：
  - 移除了原先直露在板子左侧的歌词纸条。歌词纸条现在已成为“童谣预言”线索节点的专有详情子内容，点击“童谣预言”卡片即可在右侧详情栏进行精美查阅，并与嫌疑人遇害状态保持划线同步。
- **等比自适应展现**：
  - 引入 `ResizeObserver` 动态计算缩放比例，自动将 1000px * 600px 的整张连线面板以 scale 缩放并完全垂直水平居中收纳至容器中，彻底隐藏滚动条，且大屏下最大宽度支持扩展至 `1320px`。
  - 拖拽运动矢量经过 scale 除数修正，使在大头针拖曳时的光标与卡片位置完全 1:1 精确跟随，拉扯感灵敏丝滑。
- **章节菜单回顾与方括号文本着色**：
  - 在侦查标题栏右侧增加章节菜单，支持查看侦查中与已通关的章节；回顾历史章节时一次性显现文本并挂起播放器操作。
  - 动态格式化 `[...]` 高亮金黄色文本并强制不换行，保证排版整齐。
- **玩偶小兵右上角碎裂彩蛋**：
  - 取消了原先的玩偶产量倍率面板，将小兵玩偶作为 Outline SVG 彩蛋融入到嫌疑人卡片右上角（遇害后碎裂），且全局产量倍率与进度条被完美整编至右侧栏的“案情侦破进度统计”看板中。
- **物证被动挂机升级整合**：
  - 移除了右侧简易的物证升级列表，将其与线索墙物证（便利贴）节点的详情牛皮纸袋板块进行整合。点击墙上物证即可在右侧详情栏直接查看并消费 DI 对物证执行分析升级，实时增幅挂机加成。

### [v1.1.0] - 2026-06-05
> **本次更新重点：官方域名上线、客户端自更新及 16:9 比例重塑**

#### 🚀 新增特性 (Features)
- **章节菜单回顾功能**：
  - 在侦查工作台标题栏增加“章节”选择下拉框，允许玩家在侦查进行中随时切回已通关的章节进行内容回顾。
  - 回顾历史章节时整章段落一次性全部显现，同时冻结并隐藏打字机打印、暂停/播放等常规侦查控制面板，并提供“返回当前最新章节”的快捷操作按钮。
- **方括号文本特殊着色与防折行**：
  - 对小说现场侦查与书架双语阅读正文中的 `[...]` 括号文本进行动态正则捕获与样式替换。
  - 自动渲染为护眼金黄色（`var(--palette-gold)`），并通过 CSS 的 `white-space: nowrap` 与 `display: inline-block` 属性防止该词组因排版换行而被截断分成两行。
- **阅读自动播放与暂停机制重塑**：
  - 分离为独立的 **“▶ 自动播放”** 与 **“⏸ 暂停”** 双按钮操作。
  - 真正实现“暂停”功能：当点击暂停时，不仅停止跨段落自动翻阅，还将彻底挂起并暂停当前的字词打字机流动效果；当点击自动播放时，打字机重新运行并在读毕后自动平滑翻开并解密下一段。
- **客户端自更新系统 (Tauri Auto-Updater)**：
  - 桌面客户端内增加“检查更新”专属通道，点击后会联网检测 GitHub Releases。
  - 支持在检测到新版本时展示完整的更新日志元数据。
  - 接入 **Tauri 增量静默升级与重启**，升级时提供直观的**百分比下载进度条**，网络连接异常时提供友好的出错排查说明。
- **线索墙安全重组**：
  - 解决本地安装版本中由于窗口安全沙箱限制导致 `window.open` 弹窗失效的问题，改为在应用内使用高契合度的 **软木板线索墙 Modal 遮罩层** 呈现。
  - 将“📌 案卷线索墙”按钮由全局顶栏 Header 移入特定书籍的“现场侦查”工作区内，仅在书籍解锁调查后出现，防止未解锁书籍透漏线索，并明确归属。
- **16:9 画幅比例约束**：
  - 重塑了桌面端与大屏幕下的画质呈现，强制外框为 **16:9 经典黑影剧院比例**，避免大屏拉伸影响视觉聚焦。
  - 在移动端/手机竖屏下自动回弹为 100% 流式高度，保证翻阅体验。

#### 🔧 优化与调整 (Improvements & Fixes)
- **桌面应用图标重塑**：已将项目根目录的 `idn-icon.png` 重新生成为符合 Windows (.ico), macOS (.icns) 和 Linux 平台要求的全套 Tauri 原生桌面端安装与运行图标。
- **域名变更**：将所有在线游玩链接升级为官方主域名：**[idn.fivood.com](https://idn.fivood.com)**。
- **网络鲁棒性**：为更新检测机制增加网络超时/中断友好诊断，免去原生报错给玩家带来的困惑。

### [v1.0.0] - 2026-06-05
> **本次更新重点：多案卷平行选择、华生医生关联清理、自适应双色主题及桌面端底座搭建**

#### 🚀 新增特性 (Features)
- **多案卷分组管理**：
  - 集成编译 5 部大型侦探案卷（包含《无人生还》与霍桑探案集 4 部作品）。

  - 引入 `@tauri-apps/cli` 底座，配置生成各系统原生安装包的流水线。

  - 替换了原版手动购买段落，改为随时间自动解锁字句并具有手速微调面板。

#### 🔧 优化与调整 (Improvements & Fixes)
- **去 Watson 化**：原有的“华生日记”重构为“侦查助手日记 (Assistant Journal)”，实现兼容性自动存档合并过渡。

- **精度修复**：修复了初始速率 `0.005` 导致面板上 DI 与效率双双显示为 `0` 从而产生游戏卡死错觉的严重精度截断问题。
